# Voidstone Studio E-Commerce Platform

## Microservices Architecture
- API Gateway
- Auth Service
- User Service
- Product Service
- Appointment Service
- Notification Service

## Tech Stack
- Frontend: React TypeScript + Material-UI
- Backend: Node.js TypeScript
- Databases: PostgreSQL, MongoDB
- DevOps: Docker, Jenkins, Prometheus, Grafana

## Getting Started
1. Run docker-compose up -d
2. Install services dependencies
3. Start services in order: discovery ? gateway ? auth ? others
